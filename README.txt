Ceci est le fichier README.txt demandé dans l'énoncé de l'activité du cours OpenClassrooms "Git & GitHub".

Dans ce répertoire je vais ajouter les fichiers sources d'un projet Java que j'avais réalisé dans le cadre de mes études.
Ce projet avait pour objectif de créer un modèle (très approximatif) représentant le comportement de cellules saines ou cancéreuses, mises en culture dans une boîte de Pétri.
Pour ce faire j'avais créé différents objets participant au phénomène : le milieu, la cellule, la plage de cellules et la drogue antimitotique. Il m'a également fallu créer d'autres objets destinés à récupérer les paramètres que l'utilisateur voulait utiliser pour la simulation mais aussi à modéliser graphiquement l'évolution des différentes populations de cellules.
